This is the link to the competition on kaggle : https://www.kaggle.com/c/rossmann-store-sales
The original dataset provided by kaggle is downloaded from : https://www.kaggle.com/c/rossmann-store-sales/data

The leaderboard for this competition : https://www.kaggle.com/c/rossmann-store-sales/leaderboard
Our entries are submitted under the same team name (miners)

To run the project :

1. run RossmannSales.java

2. enter the experiment which you want to run

	NOTE: the detailed description of each experiment is included in the final report, Final_Report_8_Miners.pdf
	NOTE: experiment 1 and 2 take more than 6 hours to compute.
	
3. the result for that experiment run on the test data provided by kaggle will be stored in the
	submissions directory, under the name "experiment<experiment_number>_submission.csv"
	

4. Experiment 10 has been done separately in experiment_10.py To run it go to the directory where it is located,
   and enter > python experiment_10.py
